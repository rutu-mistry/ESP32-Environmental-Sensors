# ESP32-Environmental-Sensors
ESP32-S3 examples for AHT20, SHT4x and BME280 temperature, humidity and pressure sensors.

# Environmental Sensor Testing using ESP32-S3

This repository contains Arduino examples for testing environmental sensors with the ESP32-S3.

## Sensors Tested

- AHT20 – Temperature & Humidity
- SHT4x – High Accuracy Temperature & Humidity
- BME280 – Temperature, Humidity & Pressure

## Hardware

- ESP32-S3 Development Board
- Arduino IDE
- I2C Interface

## Repository Structure

- AHT20/
- SHT4x/
- BME280/

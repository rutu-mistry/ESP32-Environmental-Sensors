# AHT20 Sensor Test

## Description

This example demonstrates how to read temperature and humidity using the AHT20 sensor over I2C with an ESP32-S3.

## Connections

| AHT20 | ESP32-S3 |
|--------|----------|
| VCC | 3.3V |
| GND | GND |
| SDA | SDA |
| SCL | SCL |

## Library

- Adafruit AHTX0

## Example Output

Temperature : 28.5 °C

Humidity : 62.4 %

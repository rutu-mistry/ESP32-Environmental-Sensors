# BME280 Sensor Test

## Description

Reads temperature and humidity using the BME280 sensor over I2C.

## Connections

| BME280 | ESP32-S3 |
|--------|----------|
| VCC | 3.3V |
| GND | GND |
| SDA | SDA |
| SCL | SCL |

## Libraries

- Adafruit BME280

## Example Output

Temperature: 28.4 °C
Humidity: 63.5 %

# SHT4x Sensor Test

## Description

Reads temperature and humidity from the Sensirion SHT4x sensor.

## Connections

| SHT4x | ESP32-S3 |
|--------|----------|
| VIN | 3.3V |
| GND | GND |
| SDA | IO8 |
| SCL | IO9 |

## Library

- Adafruit_SHT4x

Temperature : 27.8 °C

Humidity : 60.7 %

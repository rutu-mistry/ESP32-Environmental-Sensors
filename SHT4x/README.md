# SHT4x Sensor Test

## Description

Reads temperature and humidity using the AHT20 sensor over I2C.

## Connections

| AHT20 | ESP32-S3 |
|--------|----------|
| VCC | 3.3V |
| GND | GND |
| SDA | SDA |
| SCL | SCL |

## Libraries

- Adafruit AHTX0

## Example Output

Temperature: 28.4 °C
Humidity: 63.5 %
